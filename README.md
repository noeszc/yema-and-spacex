## Setup

Installs the dependencies

```bash
npm install # or yarn install
```

Create a `.env` file

```bash
EXTEND_ESLINT=true
REACT_APP_GRAPHQL_API_URL=https://api.spacex.land/graphql
```

Start development

```bash
npm start # yarn start
```
